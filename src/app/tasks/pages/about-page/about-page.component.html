About This Application info

This is a To-Do List application built with Angular for the frontend and Spring Boot for the backend. The application allows users to create, edit, complete, and delete tasks. It also separates pending tasks from completed ones, making task management straightforward and efficient.

This application is designed to help you organize your tasks effectively. You can easily track your progress by marking tasks as complete or viewing your pending tasks.
